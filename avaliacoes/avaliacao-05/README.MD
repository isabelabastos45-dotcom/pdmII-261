1- Conceito
O conceito dos “jogos de linguagem” foi desenvolvido por Ludwig Joseph Johann Wittgenstein, um filósofo austríaco que atuou no campo da filosofia e da linguística durante a primeira metade do século XX. Wittgenstein propôs que uma palavra ou mesmo uma frase só tem significado dentro das "regras do jogo" linguístico que está sendo jogado em determinado momento. Isto é, depende do local e situação, ele afirma que o sentido das palavras não seria fixo nem universal, mas esta relacionado do uso dentro de um contexto específico. 

Esse tema é introduzido principalmente em sua obra Investigações Filosóficas, nesse livro, Wittgenstein utiliza os jogos de linguagem para explicar a inconstância do significado das palavras, determinados pelo contexto de atividades sociais e formas de vida; basicamente variações de linguagem estão diretamente ligados ao social de cada indivíduo.

Em outras palavras, a linguagem funciona como um jogo, onde existem regras, sejam elas implícitas ou explícitas, e seus significados surgem da prática; cada circunstância define como a linguagem deve ser usada. Por exemplo: a palavra 'classe' pode significar: um grupo de alunos no quesito educação, estrutura em programação se estamos falando de programação orientada à objeto ou categoria social, no caso da sociologia. Ou seja, o significado depende do jogo de linguagem em que está sendo usado.

2- No ensino
No ensino, os jogos de linguagem ajudam a entender que aprender não é apenas memorizar conceitos, mas participar de práticas sociais e discursivas. O aluno aprende melhor quando o conteúdo está inserido em um contexto real e a construção do significado não é dado sem que ele tenha que raciocinar até chegar a conclusão correta, ele é construído pelo uso. Ao invés de apenas definir a palavra 'variável', o professor, nesse contexto, poderia propor situações práticas, como a programação e matemática, que usufruem desse termo. Sendo assim, o estudante aprende usando o conceito, não só decorando e o processo de aprendizado vira um jogo de linguagem ativo, que estimula o pensamento e interpretação.

3- Em POO
Na programação orientada à objeto, os termos como: classe, objeto, herança e encapsulamento formam um jogo de linguagem próprio da computação.

Por exemplo:
```dart
class Carro {
    String cor;
    int ano;
}
```
Nesse contexto, a 'classe' representa um modelo ou estrutura, o 'objeto' é uma instância dessa classe. Não tendo o significado comum do dia a dia.

Ou em:
```dart
Carro meuCarro = new Carro();
```
A palavra 'classe' não tem o mesmo significado que na educação ou sociologia.

Outro exemplo pode ser:
```dart
class Animal {
    String nome;
}

Animal cachorro = new Animal();
```
A palavra 'Animal' não representa diretamente um ser vivo, mas uma estrutura de código.
